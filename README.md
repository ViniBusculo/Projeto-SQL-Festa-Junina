# Projeto SQL – Festa Junina

Este repositório contém todos os scripts SQL, incluindo criação do banco, inserção de dados, consultas, atualizações e exclusões com integridade referencial.

## Conteúdo

### DDL – Criação das Tabelas
Arquivo: `ddl_create_tables.sql`  
Define toda a estrutura do banco `festa_junina`.

### DML – Inserts
Arquivo: `dml_inserts.sql`  
Popula todas as tabelas com dados coerentes e funcionais.

### SELECT – Consultas
Arquivo: `selects.sql`  
Inclui queries com:
- `WHERE`
- `ORDER BY`
- `LIMIT`
- `JOIN`

### UPDATE
Arquivo: `updates.sql`  
Possui 3 comandos de atualização com condições.

### DELETE
Arquivo: `deletes.sql`  
Deleções seguras que respeitam integridade referencial.

## Ambiente Utilizado
- XAMPP (MySQL)
- MySQL Workbench

## Como Executar
1. Inicie o MySQL no XAMPP  
2. Abra o Workbench  
3. Execute os scripts na ordem:
   - `ddl_create_tables.sql`
   - `dml_inserts.sql`
   - `selects.sql`
   - `updates.sql`
   - `deletes.sql`
